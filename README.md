### 🎉 Instalando as dependências

```sh
npm install
```

### 🔥 Inciando a aplicacão
> Abra o terminal e execute o primeiro comando e não fecha, depois abre outro terminal e execute o segundo comando.

```sh
npm server
npm dev
```

### 🍸 Para gerar a Build

```sh
npm run build
```
